# ⚡ Offline First

This application is designed to work even when there is no internet connection.

- Orders are stored locally
- Menu is available offline
- Checkout works without internet
- Kitchen receives orders locally
- Reports are generated from SQLite
- Data can be synced when connectivity is restored
